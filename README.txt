# 番茄的隨機抽歌單網站

這是一個會直接讀取 Google 試算表的隨機抽歌網站。

## 使用方式

1. 把 `index.html` 上傳到 GitHub Pages、Netlify，或直接用瀏覽器打開。
2. 修改 Google 試算表後，重新整理網頁就會更新。
3. 試算表目前使用：
   - A 欄：歌名
   - B 欄：歌手
   - C 欄：分類
